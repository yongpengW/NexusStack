# Changelog

All notable changes to NexusStack will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-02-24

### 🎉 Initial Release

#### Added
- ✨ Complete microservices solution template
- 🏗️ Clean layered architecture (Infrastructure, Domain, Host)
- 🔄 SignalR support with auto-mapped hubs
- 📨 RabbitMQ integration for event-driven architecture
- 💾 Redis caching support
- 🗄️ EF Core with multi-database support (PostgreSQL, MySQL, SQL Server)
- 📊 Swagger/OpenAPI automatic documentation
- 📝 Serilog structured logging
- ⏰ Background job scheduling with Cron expressions
- 🐳 Docker support with Dockerfile
- 🎨 Configurable template parameters
- 📚 Comprehensive documentation
- 🛠️ PowerShell and Shell installation scripts

#### Features
- Multiple target framework support (.NET 8, 9, 10)
- Optional background services (MQ Service, Plan Task Service)
- Flexible database provider selection
- Enable/disable features via template parameters
- Production-ready project structure
- Best practices and conventions
